# node-crud
My custom CRUD app that has a front-end and back-end for full-stack development.

## Tools used
- Front-end: ReactJS, TailwindCSS 
- Back-end: NodeJS
- Database: MySQL
- Others: axios, express, react-router, cors, multer, jspdf
